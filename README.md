# Power BI E-commerce Sales Dashboard

## Project Overview
This project analyzes an e-commerce sales dataset using Power BI to identify
revenue drivers, profit trends, and customer purchasing behavior.

## Key Business Metrics
- Total Sales: 438K
- Total Profit: 37K
- Quantity Sold: 5,615
- Average Order Value: 121

## Key Insights
- Maharashtra is the highest revenue-generating state
- Clothing contributes 63% of total quantity sold
- Cash on Delivery accounts for 44% of transactions
- Printers and Bookcases generate the highest profit
- Seasonal profit fluctuations with recovery in Q1 and Q4
